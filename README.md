# AI infrastructure capital allocation framework

## Overview

This repository is a research project on how capital may move across the AI infrastructure value chain as power becomes harder to secure.

The focus is not stock picking. The project looks at physical constraints that can shape investment decisions, especially power availability, grid capacity, and other infrastructure bottlenecks.

The goal is to build a repeatable research process that uses primary sources, scenario analysis, and clear decision frameworks.

## Research question

How should long-term capital be allocated across the AI infrastructure value chain if power availability becomes the main constraint on growth?

## Research framework

Each module follows the same basic process:

1. Define the investment thesis.
2. Gather evidence from primary or authoritative sources.
3. Assess the reliability and type of evidence.
4. Separate confirmed support from unresolved questions.
5. Build a decision framework that can be reused.
6. Translate the findings into capital allocation implications.
