# AI Infrastructure Capital Allocation Framework

## Overview

This repository develops an evidence-weighted research framework for capital allocation across the AI infrastructure value chain.

It does not provide stock recommendations. The project evaluates how structural constraints, particularly power availability, grid capacity, and infrastructure bottlenecks, may influence long-term capital allocation decisions.

The objective is to build a repeatable research process that combines primary-source validation, scenario analysis, and structured decision frameworks.

## Research question

> **How should long-term capital allocate across the AI infrastructure value chain when power availability becomes the primary scaling constraint?**

## Research framework

Each research module follows the same workflow:

1. Define the investment thesis.
2. Collect evidence from primary and authoritative sources.
3. Evaluate source reliability and evidence type.
4. Identify supporting evidence and key uncertainties.
5. Build a repeatable decision framework.
6. Translate research into capital allocation implications.
